# 1.流程
## 1.1.创建Repo
在[GitHub](https://github.com/)上创建Repo并clone到本地，假设Repo名字为`git_test`。
<br><br>

## 1.2.本地修改Repo
首先进入本地文件夹并执行`pull`操作，然后即可进行本地修改：
```bash
cd git_test
git pull
```

如果想看更改了哪些文件：

```bash
git status
```
<br>

## 1.3.上传Repo
依次执行如下操作：
```bash
git add .
git commit -m "What changes have been made"
git pull
git push
```
<br>

# 2.其他
## 2.1.修改.gitignore文件不生效
进行下面操作即可解决：
```bash
git rm -r --cached .
```
